# The Valeria Project

- **Project**: **V**isibiliz**A**cion de co**L**eccion**E**s pat**R**imoniales por medio de **I**nteligencia **A**rtificial (VALERIA).

## Dependencies

- To run the Valeria project, you need to have installed the **Miniconda** package manager.
    - Download Miniconda from the official website: [Miniconda Download](https://www.anaconda.com/download).
- To create the required environment, use the provided `environment.yml` file:
    ```bash
    conda env create -f environment.yml
    ```
- Activate the environment with:
    ```bash
    conda activate valeria
    ```
- Install the dependencies listed in `requirements.txt`:
    ```bash
    uv pip install -r requirements.txt
    ```
